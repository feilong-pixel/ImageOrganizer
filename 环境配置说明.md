# ImageOrganizer 环境配置说明

## 1. 项目简介

本项目是一个基于 Python 的图片与视频整理工具。
程序会扫描源目录，优先读取图片的 EXIF 时间；如果没有 EXIF 时间，则使用文件修改时间。
然后按照“年\月\日”的目录结构，将文件移动或拷贝到目标目录。


## 2. 推荐环境

- 操作系统：Windows 10 或 Windows 11
- Python：3.10 或更高版本
- 终端：PowerShell 或命令提示符（CMD）

本项目是本地脚本工具，不依赖数据库、Web 服务或外部接口。


## 3. 必要依赖

本项目当前依赖以下第三方 Python 库：

- `Pillow`

`Pillow` 用于读取图片的 EXIF 元数据。


## 4. 创建虚拟环境

在项目根目录打开终端后，执行：

```powershell
python -m venv venv
```

在 PowerShell 中激活虚拟环境：

```powershell
.\venv\Scripts\Activate.ps1
```

如果你使用的是命令提示符（CMD）：

```cmd
venv\Scripts\activate.bat
```


## 5. 安装依赖

激活虚拟环境后，执行：

```powershell
pip install Pillow
```

可选检查命令：

```powershell
pip show Pillow
```


## 6. 验证环境

检查 Python 和 pip：

```powershell
python --version
pip --version
```

检查 Pillow 是否安装成功：

```powershell
python -c "from PIL import Image; print('Pillow OK')"
```


## 7. 项目结构说明

主要目录和文件如下：

- `main.py`
  程序入口文件
- `core/`
  EXIF 读取和日期识别逻辑
- `services/`
  文件整理逻辑
- `locales/`
  多语言提示资源，当前支持 `zh`、`en`、`ja`
- `log/`
  每次运行后生成的日志目录


## 8. 运行脚本

基本示例：

```powershell
python main.py --src D:\InputPhotos --dst D:\SortedPhotos
```

拷贝模式示例：

```powershell
python main.py --src D:\InputPhotos --dst D:\SortedPhotos --mode copy
```

日语界面示例：

```powershell
python main.py --src D:\InputPhotos --dst D:\SortedPhotos --lang ja
```


## 9. 运行参数说明

- `--src`
  源目录，必填
- `--dst`
  目标目录，必填
- `--mode`
  可选 `move` 或 `copy`，默认值为 `move`
- `--lang`
  可选 `zh`、`en`、`ja`，默认值为 `zh`


## 10. 输出与日志

程序会自动在项目目录下创建 `log` 文件夹。

每次运行会生成一个日志文件，格式如下：

```text
organize_log_YYYYMMDD_HHMMSS.txt
```

程序执行完成后，会在终端输出本次日志文件的完整路径。


## 11. 常见环境问题

### Python 无法识别

如果终端提示找不到 `python`，请确认：

- 已安装 Python
- Python 已加入系统 `PATH`

### PowerShell 无法激活虚拟环境

如果 PowerShell 阻止脚本执行，可先运行：

```powershell
Set-ExecutionPolicy -Scope Process RemoteSigned
```

然后重新执行激活命令。

### Pillow 未安装或导入失败

如果出现 `PIL` 相关错误，请执行：

```powershell
pip install --upgrade Pillow
```


## 12. 推荐使用流程

1. 在项目根目录打开终端
2. 创建并激活虚拟环境
3. 安装 `Pillow`
4. 使用 `--src` 和 `--dst` 运行脚本
5. 执行完成后查看 `log` 目录中的日志文件
